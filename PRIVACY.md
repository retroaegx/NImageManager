# プライバシーポリシー(Japanese ver)

最終更新日: 2026-03-18

この拡張機能は、ユーザーが NovelAI 上で現在表示している画像を、ユーザー自身の NImageManager サーバーへ転送するためのものです。

## この拡張機能が行うこと

この拡張機能は、以下を行います。

- 対応する NovelAI の画像ページに転送ボタンを追加します
- ユーザーが転送ボタンを明示的にクリックした場合にのみ、現在表示中の画像を読み取ります
- ユーザーが設定した NImageManager サーバーへ、選択された画像を送信します
- サーバー設定、ログイン情報、セッション情報をブラウザ内に保存し、継続して利用できるようにします

## この拡張機能が取り扱うデータ

この拡張機能は、以下のデータを取り扱う場合があります。

- **ウェブサイト上のコンテンツ**: 対応する NovelAI ページ上で現在表示されている画像。ユーザーが転送を開始した場合にのみ扱います
- **認証情報**: ユーザーの NImageManager サーバーにログインするためのログイン情報および認証トークン
- **設定データ**: ユーザーが設定した NImageManager サーバーのアドレスおよび関連する拡張機能設定

## データの利用目的

これらのデータは、以下の目的のためにのみ使用されます。

- ユーザーを NImageManager サーバーに認証するため
- ユーザーが選択した画像をそのサーバーへ転送するため
- 拡張機能の動作に必要な設定情報およびログイン状態を保持するため

## この拡張機能が行わないこと

この拡張機能は、以下を行いません。

- 分析や広告のために閲覧履歴を収集すること
- 複数のウェブサイトにまたがってユーザーを追跡すること
- ユーザーデータを販売すること
- マーケティング目的で第三者にデータを送信すること
- 対応する NovelAI ページおよびユーザーが設定した NImageManager サーバー以外の無関係なサイトで動作すること

## データ共有

データは、以下に対してのみ送信されます。

- 対応する NovelAI ページ: ユーザー操作に応じて現在表示中の画像を読み取るため
- ユーザーが設定した NImageManager サーバー: 認証および画像アップロードのため

法令に基づく場合、またはブラウザやプラットフォーム上で拡張機能を提供するために技術的に必要な場合を除き、この拡張機能がその他の第三者へデータを共有することはありません。

## データの保存

この拡張機能は、以下の情報をブラウザ内にローカル保存します。

- 拡張機能の設定
- 設定されたサーバーアドレス
- 認証に必要なログイン情報およびセッション状態

保存されたデータは、ユーザーが拡張機能のデータを削除するか、ログアウトするか、拡張機能をアンインストールするまで、ユーザーの端末上に保持されます。

## ユーザーによる管理

画像が転送されるタイミングは、ユーザーが管理します。ユーザーの操作なしに画像転送が自動で開始されることはありません。

ユーザーは、拡張機能を無効化またはアンインストールすることで、いつでも利用を停止できます。また、利用可能な場合には、ブラウザの拡張機能管理機能または拡張機能の設定画面から、保存された設定やログイン状態を削除できます。

## セキュリティ

この拡張機能は、単一用途の実現に必要な範囲に限定してデータを取り扱うよう努めています。ただし、ユーザー自身の NImageManager サーバーおよびアクセス認証情報の安全管理については、ユーザー自身の責任となります。

## 本ポリシーの変更

このプライバシーポリシーは、拡張機能の機能またはデータ利用方法に変更があった場合に更新されることがあります。最新版は、公開されているポリシー掲載場所にて確認できます。

# Privacy Policy(English ver)

Last updated: 2026-03-18

This extension transfers images that the user is currently viewing on NovelAI to the user’s NImageManager server.

## What this extension does

This extension:

- adds a transfer button on supported NovelAI image pages
- reads the currently displayed image only when the user explicitly clicks the transfer button
- sends the selected image to the NImageManager server configured by the user
- stores the configured server address and login/session information locally in the browser so the extension can continue to work between sessions

## Data this extension handles

This extension may handle the following data:

- **Website content**: the image currently displayed on the supported NovelAI page, only when the user starts a transfer
- **Authentication information**: login credentials and/or authentication tokens used to sign in to the user’s NImageManager server
- **Settings data**: the user-configured NImageManager server address and related extension settings

## How data is used

The data is used only for the following purposes:

- to authenticate the user with their NImageManager server
- to transfer the user-selected image to that server
- to keep the extension settings and login state necessary for the extension to function

## What this extension does not do

This extension does not:

- collect browsing history for analytics or advertising
- track users across websites
- sell user data
- send data to third parties for marketing
- operate on unrelated websites outside the supported NovelAI page and the user-configured NImageManager server

## Data sharing

Data is sent only to:

- the supported NovelAI page, for reading the currently displayed image in response to user action
- the NImageManager server configured by the user, for authentication and image upload

No other third-party data sharing is intended by this extension, except when required by law or when technically necessary for the browser or platform to deliver the extension.

## Data storage

This extension stores the following locally in the browser:

- extension settings
- configured server address
- login/session state required for authentication

Stored data remains on the user’s device unless the user removes the extension data, logs out, or uninstalls the extension.

## User control

The user controls when an image is transferred. No image transfer is started automatically without user action.

The user can stop using the extension at any time by disabling or uninstalling it. The user can also remove saved settings and login state through the browser’s extension management features or the extension settings, where available.

## Security

Reasonable efforts are made to limit data handling to what is necessary for the extension’s single purpose. However, users are responsible for securing their own NImageManager server and access credentials.

## Changes to this policy

This Privacy Policy may be updated when the extension’s functionality or data usage changes. The latest version will be made available at the published policy location.
